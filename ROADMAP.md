# Roadmap

## Current

- `P0`: runtime core, permissions, sessions, hooks, model driver
- `P1`: interactive CLI and REPL ergonomics

Both are now complete enough for a `v0.1.0` release.

## Next

### P2: Model And Provider Depth

- native tool-calling where providers support it
- stronger structured-output handling
- provider-specific adapters beyond generic OpenAI-compatible mode
- better real-model transcript stability
- richer execution progress visibility during turns

### P3: Product UX And Integrations

- stronger terminal presentation layer
- optional status panes / progress views
- MCP integration
- remote session support
- editor / IDE integration

## Non-Goals For v0.1.x

- React/Ink parity
- full Claude Code feature parity
- plugin marketplace or arbitrary script hooks
